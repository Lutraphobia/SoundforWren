## Module layout (framework, current and planned)

```
SoundforWren/
├── sensory_report.py          v1: MUSIC                    (shipped)
├── signal_report.py           v2: SIGNAL (ham radio modes) (planned, see ROADMAP.md)
├── speech_report.py           v3: SPEECH                   (optional)
└── ambient_report.py          v4: AMBIENT                  (optional)
```

All modules emit the same report shape: `report.md` plus `report.json` plus a heartbeat-equivalent irreducible signal, with a matching set of dark-palette visualizations.
